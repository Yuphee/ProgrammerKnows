## ProgrammerKnows
   本项目旨在收集记录从事计算机领域工作者在学习的时候设计到的其它领域的一些知识，见解，包含不仅限于计算机中的知识体系，可以当作是兴趣阅读，也可以当作是学习目前暂时记录我在学习过程中的一些经历，如果有其他人的分享，可以Pr或issue，第一个知识点还在完成中，敬请期待。
